# SuperScoala APK

Acest proiect contine o aplicatie Android pentru [superscoala.ro](https://superscoala.ro).

Scopul principal este ca platforma sa poata fi folosita mai usor pe dispozitive Android mai vechi, unde experienta din browser poate fi limitata sau mai slaba.

Aplicatia ofera:

- acces rapid la superscoala.ro direct din telefon;
- o varianta simpla de instalat prin APK;
- compatibilitate orientata catre terminale Android mai vechi.

APK-ul poate fi descarcat de aici: [GitHub Releases](https://github.com/gabbello/scoalaApp/releases).

Repository-ul include proiectul Android si configurarea necesara pentru generarea fisierului APK.
